# Rica-Coffee-Latte-SHOPBD-WEBSITE
Rica Coffee &amp; Latte SHOPBD – A premium coffee &amp; latte website crafted with modern HTML, CSS, and JavaScript. Showcases signature beverages, interactive product cart, smooth scrolling, and a stylish responsive design for an immersive user experience.

Features

Premium Coffee Showcase – Browse signature coffees and lattes with images, descriptions, and prices.
Interactive Cart – Add items to cart, remove them, and track quantities with real-time auto-calculation of totals.
Place Orders – Select multiple items, fill in customer details, and place orders stored locally.
Responsive & Modern UI – Mobile-friendly design with sleek hero section, hover effects, and smooth transitions.
About Section – Introduces the brand, mission, and coffee experience.
Footer Contact Info – Easily accessible email, phone number, and address.
Navigation & Hero Section – Quick access to menu, about, and order pages with engaging visuals.

Built With / Tools

HTML5, CSS3, JavaScript – Core web technologies for structure, styling, and interactivity.
LocalStorage – For storing cart items and order data.
Google Fonts – Poppins font for modern typography.
Responsive Design Techniques – Flexbox, Grid, and media queries for mobile-friendly layout.
